# FConline Manual — 운영기획 PRD
> 작성일: 2026-04-11 | 버전: v15.3 기준 분석

---

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | FC Online 유저가 선수 정보, 경기 분석, 전술 팁을 흩어진 곳에서 수동으로 찾아야 하는 불편함 |
| **Solution** | Nexon 공식 API + AI 기반 분석을 한 앱에서 제공하는 FC Online 전용 정보허브 PWA |
| **UX Effect** | 모바일에서 즉시 검색·분석 가능 → 게임 중 실시간 참고 가능 |
| **Core Value** | "게임하면서 바로 꺼내보는 FC Online 도우미" |

---

## 1. 현황 분석 (As-Is)

### 1.1 앱 구조 요약

| 탭/섹션 | 주요 기능 |
|---------|----------|
| **홈** | 공식 이벤트 캐러셀, 오늘의 팁, 컨트롤러 사용 가이드, 커뮤니티 미리보기 |
| **선수** | 이름/포지션 검색, 포지션별 인기 TOP 5, 선수 상세 (능력치·강화·적응도), AI 선수평가, 리뷰 |
| **분석** | 구단주 닉네임 검색 → 1:1 공식경기 분석, 볼타 분석, 공격패턴·주요선수 TOP 5 |
| **커뮤니티** | 게시글·댓글 작성, 공유 링크 |
| **마이** | 다크모드 토글, 앱 문의, 버전 노트 |

### 1.2 기술 스택

- **Frontend**: Next.js (App Router) + Tailwind CSS + Framer Motion
- **Backend**: Supabase (DB + Auth) + Nexon Open API
- **배포**: Vercel (fconlinemanual.vercel.app)
- **PWA**: 모바일 홈화면 설치 지원

### 1.3 릴리스 속도

v11.x → v15.3까지 약 40회 이상 릴리스. **평균 주 1~2회** 배포. 주요 패턴:
- UX 버그픽스 (safe-area, 다크모드 깜빡임 등) 비중 높음
- 기능 추가 후 빠른 후속 polish 패턴

## 2. 사용자 분석 (Who)

### 페르소나

| 페르소나 | 특성 |
|---------|------|
| **라이트 유저** | FC Online 입문자, 어떤 선수를 사야 할지 모름. 인기 TOP 5 카드 주로 활용 |
| **코어 유저** | 특정 선수 능력치 비교, 강화단계별 수치 확인, 경기 분석 탭 적극 사용 |
| **대전 분석러** | 상대 구단주 닉네임 검색해서 공격패턴·주요선수 미리 파악 |

### 핵심 JTBD (Jobs-To-Be-Done)

1. "이 선수 지금 살 가치 있어?" → 선수 능력치 + AI 평가 + 커뮤니티 의견
2. "상대가 어떻게 플레이하나?" → 구단주 경기분석
3. "지금 어떤 이벤트 있어?" → 홈 이벤트 캐러셀
4. "볼타에서 어떤 코치 쓰지?" → 볼타 훈련코치 가이드

## 3. 경쟁 분석

| 서비스 | 특징 | FConline 차별점 |
|-------|------|----------------|
| FC온라인 공식 사이트 | 공식 정보만, 분석 없음 | 경기 분석 + AI 평가 |

| 게임 카페/디스코드 | 텍스트 위주, 검색 불편 | 구조화된 검색·비교 |

**핵심 차별화**: 모바일 PWA + Nexon API 실시간 데이터 + AI 분석 통합

## 4. 운영기획 — 개선 기회 영역

### 4.1 즉시 개선 (Quick Wins, 1~2주)

| # | 개선 항목 | 근거 | 예상 효과 |
|---|---------|------|---------|
| Q1 | **선수 비교 기능** | 현재 선수 상세를 탭 전환으로 비교해야 함 | 구매 의사결정 속도 향상 |
| Q2 | **검색 히스토리 저장** | 매번 재검색 필요 | 재방문 UX 개선 |
| Q3 | **홈 개인화 (최근 본 선수)** | 홈에 개인화 콘텐츠 없음 | 체류 시간 증가 |
| Q4 | **선수 즐겨찾기** | 자주 보는 선수 바로 접근 | 재방문율 향상 |

### 4.2 중기 기획 (1~2개월)

| # | 기획 항목 | 설명 | 기술 요구 |
|---|---------|------|---------|
| M1 | **경기 메타 분석 대시보드** | 포메이션 트렌드, 포지션별 선호 선수 집계 | Nexon API 집계 + Supabase 저장 |
| M2 | **AI 전술 추천** | 내 구단주 닉네임 입력 → AI가 약점 분석 후 보완 선수 추천 | Claude API 연동 강화 |
| M3 | **시즌 카드 비교** | 같은 선수의 시즌별 능력치 변화 그래프 | 시계열 데이터 저장 |
| M4 | **커뮤니티 강화** | 게시글 카테고리화, 핫게 랭킹, 이미지 첨부 | Supabase Storage |
| M5 | **푸시 알림 (PWA)** | 이벤트 시작/종료, 인기 선수 가격변동 알림 | Web Push API |

### 4.3 장기 로드맵 (3개월+)

| # | 방향 | 설명 |
|---|------|------|
| L1 | **소셜 기능** | 팔로우, 내 팀 공유, 구단주 랭킹 프로필 |
| L2 | **클랜/팀 기능** | 클랜원 경기 분석 공유 |
| L3 | **마켓 가격 추적** | 선수 카드 시세 변동 알림 |
| L4 | **다국어 지원** | 영어/일어 → 글로벌 FC Online 유저 확보 |

## 5. 기술 부채 및 리스크

| 항목 | 현황 | 권고 |
|------|------|------|
| **Nexon API 의존도** | 모든 경기/선수 데이터가 외부 API에 의존 | 캐시 레이어 강화, API 다운 시 fallback UI |
| **캐시 전략 불일치** | TTL 기준이 기능마다 다름 (v15.3에서 정리 중) | 통합 캐시 전략 문서화 |
| **유니코드 하드코딩** | `\uC120\uC218` 등 이스케이프 문자 코드 내 산재 | 상수 파일로 일원화 |
| **테스트 부재** | 빠른 릴리스 대비 테스트 코드 없음 | 주요 API route 단위 테스트 도입 |
