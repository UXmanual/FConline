Usage Guide

Primary Action

#256EF4를 링크, 포커스 포인트, 주요 액션 강조색으로 사용합니다.

Text Hierarchy

#1E2124는 제목, #464C53은 본문, #8A949E는 보조 텍스트에 사용합니다.

Surface System

#F4F5F6는 앱 배경, #FFFFFF는 카드와 메인 콘텐츠 배경에 사용합니다.

Separators

#E6E8EA는 구분선, #58616A는 입력 테두리처럼 더 강한 경계에 사용합니다.
